Role User Allocation


This guide provides step-by-step instructions on how to allocate a role to a user in UNCIA Studio. It includes information on how to navigate to the Role User Allocation page, select the desired role, and link it to a user. This guide is essential for anyone who needs to assign roles to users in UNCIA Studio.

Raghav Bansal | 5 steps | 27 seconds

Azure Websites

1. The list of approved Roles is displayed on screen.

Click on the "Edit" option for the respective role to allocate to a user.

Prerequisite: Role and User creation in Routine Setup --> Role and Routine Setup --> User respectively

2. Click "+ Link User" in the User Mapping window to link a user.

3. Check the "Select" field for the respective user to Link it to the role.

4. Click "confirm selection".

5. Click "Submit"
